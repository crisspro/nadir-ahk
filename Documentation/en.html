NadIR-AHK help

NadIR-AHK is a script to use NadiR VST interface with screen readers.
Should working with any DAW. Tested with:

  • NadIR v1.0 32bits
  • Reaper v6.11 64bits and Audacity v2.4.2 (Must be work with aother sequenceers/editors)
  • NVDA, Jaws for Windows, Sapi5
  • windows10 64bit

GPL-3.0 License
Source code and last releases: https://github.com/crisspro/nadir-ahk

Usage

  1. Run NadIR-AHK
  2. Open NadIR VST plugin.
  3. Use the hotkeys of script.
  4. You can close the script on taskbar.

About NadIR

NadIr is a free vst by igniteamps to Mac and Windows of 32 and 64 byts. With this plugin you can load impulses responses to guitar or guitar bass. A impulse response is a audio file (prefer WAV), this file have equalization parameters to emule traditional cabinets of famous models or bands. So if you put Nadir after a distortion plugin, you will get a natural† a exelent sound. You can get so much impulses responses packs free in the internet. Visit: http://www.igniteamps.com/

Functions

  • Low pass
  • High pass.
  • Paner
  • Valance
  • Delay
  • Mode (mono, doble mono, stereo ).
  • Quality (low, normal, High and extreme).
  • Power 1 (on/off button to channel 1)
  • Power 2 (on/off button to channel 2)
  • Bypass (on/off NadIR button)

You can change these parameters from sequeencer.

Stereo mode

NadIr can create a stereo effects, with it you get a effects like play with two guitars at the same time. Also can select different impulses to right and left. After loaded a impulse to first channel (channel for default), to enable stereo effect, you have to do:

  1. Press ‚Äúm‚Äù and select stereo.

  3. Press ‚Äú2‚Äù to load the impulse response.
  4. Go to Sonar and in the sequeencer put delay 1 or delay 2 at 20 ms and play your guitar to listen.

    5. Hotkeys

    Hotkey Description
    1 Load a impulse to channel 1 (channel for default)
    F Next impulse to channel 1
    D go Back at impulse to chanel 1.
    S List of Channel 1 impulses
    M Mode menu.
    C Quality menu.
    Alt+1 Enable and disable channel 1 (it is enabled for default).
    Alt+2 Enable and disable channel 2 (it is disabled for default).
    2 Load impulse to channel 2
    K Next impulse to channel 2
    J Go back at impulse to chanel 2
    L List of channel 2 impulses
    Alt+3 Off/on NadIR (it is on for default).
    F1 Open help documentation
    CTRL+q close the script
