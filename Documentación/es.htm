Ayuda de NadIR-AHK

 

NadIR-AHK es un script para usar el plugin VST NadIR con lectores de pantalla.

-          Compatible con NadIR v1.0

-          Probado en Reaper y Audacity (también pudiera funcionar en otros secuenciadores/editores de audio)

Instrucciones

 

1.       Para usar el script, EJECUTA NadIR-AHK.

2.       Carga Nadir VST en tu secuenciador y ya podrás comenzar a trabajar.

3.       Para cerrar el script busca NadIR-AHK en la barra de tareas, haz clic derecho y se abrirá el menú con diferentes opciones, allí está la opción para salir.

 

Recomendaciones

 

·         La ventana de NadIR debe tener el foco para que funcione el script, si usas Reaper procura activar la ventana flotante.

·         Dado que el script detecta el título de las ventanas, si por casualidad quieres usar alguna ventana diferente al plugin que tenga en su título NadIR, procura desactivar el script para que no interfiera con dicha ventana.

·         Si al cargar un impulso vuelve a abrirse la ventana para buscar el archivo, solo presiona escape.

 

 

Sobre NadIR.

 

NadIR es un plugín vst gratuito desarrollado por igniteamps para Mac y Windows de 32 y 64 byts, cuya función es cargar impulsos de respuesta para instrumentos como bajos o guitarras. Un impulso es un archivo (preferentemente en formato WAV) que posee parámetros de ecualización predefinidos. Esta ecualización emula los altavoces de un amplificador de guitarra o bajo tradicionales, de manera que al añadirlo después de un plugín de distorsión, se consigue un efecto natural.

Puedes conseguir variados packs de impulsos gratuitos navegando por la red.

 

Visita: http://www.igniteamps.com/

 

 

Funciones de NadIR.

 

·         Filtro pasa altos.

·         Filtro pasa bajos.

·         Panoramización.

·         Balance.

·         Delay.

·         Modos (mono, doble mono, estéreo).

·         Calidad (baja, normal, alta, extrema), a mayor calidad utiliza más recursos del PC.

·         Botón de encendido y apagado para el canal 1.

·         Botón de encendido y apagado para canal 2.

·         Botón de encendido y apagado total .

 

Nota:

Todos estos parámetros pueden ser cambiados desde el secuenciador como ocurre con la mayoría de efectos.

 

 

Cargando un impulso.

 

Cargar un impulso es muy sencillo:

a.       Presiona 1 Se abrirá una ventana para buscar el impulso, una vez lo encuentres le das a enter y listo.

b.       Si hay mas de un impulso en la misma carpeta, puedes presionar k para cambiar al siguiente impulso y con j vas al impulso anterior. 

 

 

Activando el modo estéreo

 

NadIR permite usarlo en modo estéreo, esto significa que se puede lograr un sonido envolvente con una sola guitarra, ya que sonará por ambos auriculares, simulando ser dos guitarras al mismo tiempo. A su vez, es posible cargar un impulso diferente para cada lado del campo estéreo (izquierdo y derecho).

Una vez cargado el impulso para el canal 1 (canal por defecto apenas se abre NadIR), si quieres activar el efecto estéreo has lo siguiente:

 

a.       Desplegar el menú modo presionando m y escoger la opción estéreo.

b.       Encender el segundo canal presionando alt+2.

c.       Cargar el impulso presionando 2.

d.       Desde el secuenciador poner el delay2 a las 20ms (20 milisegundos).     

 

 

Comandos de teclado.

 

Comando

Descripción

1

Abre la ventana para cargar los impulsos en el canal 1 (canal por defecto).

f

Cambia al siguiente impulso del canal 1.

d

Cambia al  impulso anterior del canal 1.

s

Despliega la lista de impulsos en la carpeta del canal 1

m

Despliega un menú para cambiar entre los modos mono, doble mono o estéreo.

c

Despliega un menú para cambiar la calidad de sonido.

Alt+1

Activa y desactiva el canal 1 (está activo por defecto)

Alt+2

Activa y desactiva el canal 2 (está desactivo por defecto).

2

Abre la ventana para cargar impulsos en el canal 2.

k

Cambia al siguiente impulso del canal 2.

j

Cambia al impulso anterior del canal 2.

L

Despliega la lista de impulsos en la carpeta del canal 2

Alt+3

Enciende y apaga Nadir (está encendido por defecto).
